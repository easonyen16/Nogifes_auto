
■ USMファイル(.usm)やCRIDファイルの分離方法

  USMファイルやCRIDファイルをcrid.exeにドラッグ＆ドロップと、映像と音声が分離されます。
  複数ファイルの分離にも対応してます。


■ 仕様

  全ての映像データと音声データが暗号化されているものとして扱います。
  そのときに使う復号鍵は、PSO2のものですが、オプション指定することで鍵を変えることもできます。

  鍵が CF222F1FE0748978 の場合、
  crid.exe -a E0748978 -b CF222F1F -o "出力ファイル名.demux" "入力ファイル名.usm"


■ 注意事項

  Program Files (x86) フォルダ内のUSM、CRIDファイルを直接ドラッグ＆ドロップすると
  管理者権限の関係で、同フォルダ内にファイルが作成されないことがあります。
  そのときは一度、対象のファイルをcrid.exeがあるフォルダにコピーしてから分離を行うと正常に出力されると思います。
